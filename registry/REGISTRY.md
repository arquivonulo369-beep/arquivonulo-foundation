# ArquivoNulo Protocol Registry

Canonical registry for the ArquivoNulo Integrity Protocol (ANIP).

This registry maintains the official specifications that define the
deterministic governance and cryptographic validation framework of
ArquivoNulo Global.

All finalized specifications are immutable once published.

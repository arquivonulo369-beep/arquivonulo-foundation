# ArquivoNulo Improvement Proposal Process (ANIP)

## Purpose

The ANIP process defines how new protocol proposals are submitted,
reviewed, and adopted within the ArquivoNulo ecosystem.

This structure ensures deterministic governance of protocol evolution.

---

## Proposal Stages

1. Draft
Initial proposal submitted by an author.

2. Review
Technical analysis and feedback.

3. Accepted
Proposal is approved and becomes part of the protocol registry.

4. Implemented
Specification is used in production systems.

---

## File Naming Convention

ANIP-XXXX.md

Example:

ANIP-0001.md  
ANIP-0002.md

# Cryptographic Profile — ANP-004

SHA3-384

Purpose:
• Canonical artifact sealing
• Integrity verification
• Aggregate state fingerprinting

## Signature Scheme
Ed25519 (EdDSA)

Purpose:
• Institutional authorship proof
• Detached signature validation
• Non-repudiation of releases

## Aggregate Coordination
Aggregate Commit Boundary (ACB)

• Multi-agent proof aggregation
• Deterministic admissibility
• Runtime enforcement

## Trust Model

Integrity must be computationally verifiable.

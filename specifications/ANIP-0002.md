# ANIP-0002
Secure Trust Hash Standard (STH)

Status: Draft  
Author: ArquivoNulo Global  
Category: Cryptographic Integrity  
Created: 2026

---

## Abstract

This document defines the Secure Trust Hash (STH) standard used by the
ArquivoNulo Integrity Protocol.

STH establishes the canonical cryptographic hashing mechanism used
to anchor artifacts, protocol specifications and system states.

---

## Motivation

Cryptographic anchoring is required to ensure that artifacts
remain verifiable and immutable over time.

ArquivoNulo adopts SHA3-384 as the canonical hashing algorithm
for all Secure Trust Hash operations.

---

## Hash Algorithm

The canonical hashing algorithm is:

SHA3-384

Properties:

• collision resistance  
• cryptographic stability  
• long-term security margin  

---

## Hash Representation

Hashes must be represented as lowercase hexadecimal strings.

Example:

ffe902fbcf57a1ea8bfc0e2bea844534ac4253559d5e97961f265e27e72a5bc15c27b9185bbc45b39cbe7dfe59794ca9

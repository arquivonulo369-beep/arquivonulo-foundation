# ArquivoNulo Security Policy

This document describes how to report security vulnerabilities
related to the ArquivoNulo Protocol.

---

## Supported Versions

Currently supported versions:

v2.0
v2.0-hardened

---

## Reporting a Vulnerability

If you discover a vulnerability in the ArquivoNulo protocol,
please report it responsibly.

Contact:

security@arquivonulo.org

Please include:

• detailed description  
• steps to reproduce  
• potential impact

---

## Disclosure Policy

ArquivoNulo follows a responsible disclosure model.

Reported vulnerabilities will be reviewed and addressed before
public disclosure when possible.

---

## Cryptographic Integrity

The protocol relies on:

• SHA3-384 hashing  
• deterministic artifact verification  
• immutable registry records
